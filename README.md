# homebrew-tap

Homebrew tap for projects by [Ben Word](https://github.com/retlehs).

## Install

```bash
brew tap retlehs/tap
```

## Formulae

| Name | Description | Install |
|---|---|---|
| [quien](https://github.com/retlehs/quien) | A better WHOIS lookup tool | `brew install retlehs/tap/quien` |

## Casks

| Name | Description | Install |
|---|---|---|
| [somafm-player](https://github.com/retlehs/somafm-macos-player) | Native macOS menu bar player for SomaFM | `brew install --cask retlehs/tap/somafm-player` |
